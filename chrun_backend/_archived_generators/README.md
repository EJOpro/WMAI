# 📦 아카이브: 더미 데이터 생성 스크립트

이 폴더에는 이탈률 분석을 위한 **더미 이벤트 데이터 생성 스크립트**들이 보관되어 있습니다.

## 📂 포함된 파일

### 1. `generate_2025_events.py`
- 2025년 전체 이벤트 데이터 생성
- 사용자 활동 로그 시뮬레이션

### 2. `generate_2025_natural_events.py`
- 자연스러운 패턴의 2025년 이벤트 생성
- 실제 사용자 행동 패턴 반영

### 3. `generate_dummy_events.py`
- 기본 더미 이벤트 데이터 생성
- 테스트용 간단한 데이터셋

### 4. `generate_story_events.py`
- 스토리 기반 이벤트 데이터 생성
- 특정 시나리오를 따르는 사용자 여정 시뮬레이션

### 5. `generate_validation_data.py`
- 검증용 데이터 생성
- 이탈률 예측 모델 테스트용

## ⚠️ 주의사항

이 파일들은 **개발 및 테스트 목적**으로만 사용되었으며, 현재 프로덕션 코드에서는 사용되지 않습니다.

## 🆕 최신 데이터 생성 방법

**RAG 시스템용 위험 문장 데이터**를 생성하려면:

```bash
cd chrun_backend
python seed_vector_db.py
```

이 스크립트는 ChromaDB에 실제 커뮤니티 위험 문장 패턴을 저장합니다.
